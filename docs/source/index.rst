Welcome to Python Language Server
=================================

**Python Language Server** is a pure-python implementation of the `Language Server Protocol`_. 
Check out the LanguageServer for a high-level API for creating language servers, protocol for a jsonrpc implementation, or lsp for the JSON protocol definition.


.. note::

   This project is under active development.
